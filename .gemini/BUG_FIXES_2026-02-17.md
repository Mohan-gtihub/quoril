### 3. ✅ Incorrect Filter/Reduce Logic (Lines 368-390)
**Problem:** Code used `.filter()` but tried to return and sum numbers, which doesn't work. The filter was also using `Math.round()` on the whole chain instead of just the final result.

**Fix Applied:** Changed from `.filter()` with returns to proper `.filter()` + `.reduce()` pattern:

**Before (BROKEN):**
```typescript
const totalSecondsTodayFromTasks = Math.round(
    activeTasks
        .filter(t => {
            // ... checks
            if (overlapsToday) {
                let taskSeconds = t.actual_seconds || 0
                // ... calculation
                return taskSeconds  // ← Can't return number from filter!
            }
            return 0
        }) / 60)  // ← Division on filtered array, not a number!
```

**After (FIXED):**
```typescript
const totalSecondsTodayFromTasks = activeTasks
    .filter(t => {
        // ... checks
        return taskStart <= todayEnd_for_calc && taskEnd >= todayStart_for_calc
    })
    .reduce((acc, t) => {
        let taskSeconds = t.actual_seconds || 0
        // ... calculation
        return acc + taskSeconds
    }, 0)
```
